Meu Primeiro Projeto

Este projeto foi desenvolvido como parte dos meus primeiros estudos em HTML, com aproximadamente uma semana de aprendizado na linguagem.

A aplicação consiste em um site institucional fictício para uma faculdade chamada UNES, criado com o objetivo de praticar a estruturação de páginas web utilizando HTML puro.

O projeto contém três páginas principais:

1 Página Inicial – Apresentação da faculdade e introdução ao site.

2 Quem Somos – Seção dedicada à história e aos valores da instituição fictícia.

3 Contato – Página com formulário contendo campos de texto e opções de seleção para simular a interação com o usuário.

Este projeto possui finalidade educacional e foi desenvolvido para consolidar conceitos fundamentais como:

I. Estruturação com tags HTML

II. Organização de conteúdo em múltiplas páginas

III. Criação de formulários

IV. Navegação entre páginas

Por ser meu primeiro projeto, ele representa o início da minha jornada no desenvolvimento web e marca minha evolução prática na área de programação.
